The StarLight IL weaver performs code injection at the IL level. It adds IL instructions, the filter actions and some management functions, at the start of a method and it can replace calls with outputfilters. Besides the methods, it can also add internals and externals to a type.

Code injection is only performed when the analysis tools indicate that the weaver has to generate code for a certain type or method. Methods with no filter actions are left untouchted by the StarLight weaver.
