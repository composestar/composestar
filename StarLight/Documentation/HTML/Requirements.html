Requirements

The StarLight version of ComposeStar requires:

  • Java Runtime Environment 1.4.2 or higher (download)
  • Microsoft .NET Framework version 2.0 (included with Visual Studio .NET 2005) (download)
  • Optional, but recommended: Microsoft Visual Studio .NET 2005, standard of higher (not the express editions)

Important: You should be at least a Windows Power User on your system in order to install ComposeStar StarLight.

The StarLight installer will create a folder at a user specified location, adds a StarLight.target file and MS Build tasks to the c:\program files\msbuild\ folder. It will also add information into the registry at the HKLM\Software\ComposeStar\StarLight\(version) hive. Make sure your account has the permissions to execute these actions.
