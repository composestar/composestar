Requirements

The StarLight version of ComposeStar requires:

  • Java Runtime Environment 1.4.2 or higher (download)
  • Microsoft .NET Framework Version 2 (included with Visual Studio .NET 2005) (download)
  • Optional, but recommended: Microsoft Visual Studio .NET 2005, standard of higher (not the express editions)

Important: You should be at least a Windows Power User on your system in order to install ComposeStar StarLight.
