Wokring with a concern project.

ComposeStar StarLight uses a special project for the seperation of the concerns and the base code. This cpsproj file is used as input for MSBuild and a Visual Studio Project Service allows the developer to open this cpsproj in Visual Studio 2005.

All the normal operations you also perform on languages like C# or VB.NET, apply to the concern project. You can add concerns to the project, use the build command to weave the concerns into the base code, or use the property page of the project to change the settings.
