Concern project properties

The concern project properties are divided into general and build property groups.

General project properties

  • Assembly Name, name of the assembly (only used if the concern project contains CSharp, JSharp, or VB.NET code)
  • Default Namespace, default namespace used for newly created items
  • Output Type (Console Application, Class Library, Windows Application), output type of the concern project if it contains code
  • Repository filename, name of the xml file the StarLight compilation information is stored in


Build project properties

  • Debug level (Error, Crucial, Warning, Information, Debug), debug level of the StarLight compiler
  • Verify IL (True, False), true if the assemblies should be verified using the PeVerify tool (SDK required) after weaving, false otherwise
