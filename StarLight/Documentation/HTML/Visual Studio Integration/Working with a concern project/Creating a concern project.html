Creating a concern project

You can create a new StarLight concern project using the New Project dialog (File menu > New > Project). Select the 'StarLight' project type, choose a StarLight template and fill out the project settings.




There are four StarLight templates available:
  • StarLight project, this project can only contain concerns (.cps files)
  • CSharp StarLight project, this project can contain concerns (.cps files) and CSharp source code (.cs files)
  • JSharp StarLight project, this project can contain concerns (.cps files) and JSharp source code (.jsl files)
  • VB.NET StarLight project, this project can contain concerns (.cps files) and VB.NET source code (.vb files)

After creating a concern project you can modify the concern project properties, add references to basecode assemblies or add concerns.
