Referencing basecode assemblies

The concerns specified in the concern project (Concerns node in the Solution Explorer example) are only imposed on assemblies/projects referenced in the concern project.

Use the Add Reference... option in the Project menu item to display the Add Reference dialog. Select either assembly files or project files to add to the concern project.

When you build the concern project, all the referenced assemblies are copied to the output folder and are used for weaving. Referenced project are build first and referenced assemblies in a referenced project are also copied and weaved upon.

Note: Concerns are only imposed on referenced assemblies that have their Copy Local attribute set to True.
