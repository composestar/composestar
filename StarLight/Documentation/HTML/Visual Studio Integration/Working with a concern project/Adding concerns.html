Adding a concern

A new concern can be added to a concern project using the Add New Item dialog (Project > Add New Item). One of the templates available is called Concern.



The created concern file contains a template for a Composition Filter concern. More information about the Composition FIlter Language can be found on the ComposeStar website.



A concern file also has properties, which can be changed using the Properties dialog for that file. To mark a file as concern file its Build Action should be set to Concern.

Note: A concern file can easily be excluded from the StarLight compilation process by setting its build action to None
