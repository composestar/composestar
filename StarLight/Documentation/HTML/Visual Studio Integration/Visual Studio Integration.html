Visual Studio Integration

ComposeStar StarLight is integrated into Visual Studio 2005 in a number of ways.

  • First, it offers a project service to open and use StarLight project (cpsproj) files in Visual Studio.
  • Second, there is a language service to enable syntax highlighting and IntelliSense for concern files (files with the cps extension).
  • Third, MSBuild is used to analyze and weave the concerns.
