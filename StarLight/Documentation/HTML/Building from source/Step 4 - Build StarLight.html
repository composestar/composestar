Step 4 - Build StarLight

To open the StarLight build menu, follow the steps bellow:

  1. Open a Visual Studio 2005 command prompt
  2. Go to the StarLight source folder
  3. Set the JDK path: SET JAVA_HOME=<JDK root folder> (long folder names or folder names with spaces have to be set in the 8.3 format using the ~1)
  4. Run the batchfile menu.bat
