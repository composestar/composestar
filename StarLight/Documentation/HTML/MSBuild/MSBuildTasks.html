MSBuild tasks

Currently, the following tasks are available for the StarLight compilation;

Task name Description
CPS Parser
Parses the concern files and determines if the analyzer has to perform call analysis and determine the types defined for internals and externals in the concern file.
IL Analyzer
Scans the .NET assemblies and retrieves the code model and call analysis from the assemblies. This information is needed for the Java part to reason about the code.
Master Caller
Executes the Java Master to perform the filter reasoning and filter generation.
IL Weaver
Uses the output of the Java part to inject the filters into the assemblies.
